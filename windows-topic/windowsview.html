Microsoft Windows

Microsoft Windows (Windows) is a family of graphical user interface-based operating systems developed and distributed by Microsoft. It includes several lines of operating systems, each of which serves a certain part of the computer industry. Current Windows lines include Windows NT, Windows Embedded Compact and Windows Phone; they may include subfamilies, for example Windows Embedded Compact (Windows CE) or Windows Server. Discontinued Windows family lines include Windows 9x, Windows Mobile, and Windows Phone. Microsoft introduced an operating environment called Windows on November 20, 1985 as a graphical operating system interface for MS-DOS in response to growing interest in graphical user interfaces ( GUI).Microsoft Windows gradually dominated the world personal computer market with more than 90% market share, surpassing Mac OS, which was introduced in 1984. However, since 2012, its market share has lagged behind that of Mac OS. For marketing purposes

Microsoft, the developer of Windows, has registered multiple trademarks, each intended to represent a family of Windows operating systems targeted at a specific part of the computer industry. As of 2014, the Windows families under active development are:

- Windows NT:

Started as an operating system family with Windows NT 3.1, an operating system for servers and workstations. The latest version is Windows 11. Its main competitor is Apple Inc.'s macOS operating system. This family is divided into three sub-families of operating systems released at almost the same time and sharing a common core (Core).

-Windows IoT:

Initially, Microsoft developed Windows CE as a general-purpose operating system for all devices with limited resources (hardware configuration). Later, Windows CE was renamed Windows Embedded Compact and folded under the Windows Compact brand which also included Windows Embedded Industry, Windows Embedded Professional, Windows Embedded Standard, Windows Embedded Handheld and Windows Embedded Automotive.[11]

-Windows 10X:

Operating system for ARM devices, later discontinued by Microsoft.

-Windows Server:

Operating system for servers. The latest version is Windows Server 2022. Unlike consumer versions, operating systems in this family are named according to the rule (Windows Server + release year name). Their main competitor is the Linux operating system.

-Windows PE:

A lightweight version of Windows created to operate as a live operating system, used to install Windows on brand new computers (especially when multiple computers must be installed at the same time) , or used for recovery or error correction purposes. The latest version is Windows PE 10.0.10586.0.
